RHUMC

Mathematics Conference Scheduling Software for Rose-Hulman

Undergraduate Math Conference Scheduling Software

This is the GitHub Page for the Rose-Hulman Undergraduate Math Conference Scheduling Software (RHUMC). It is written in Python using the Django web framework and designed to serve as free and lightweight schedule management software for academic conferences and the likes. It was specifically designed for Rose-Hulman's Undergraduate Math Conference, but it should suitable for any sort of conference gathering.

Features

The main features include:

  • Easy student/faculty registration via web forms
  • Ability to create additional pages (for information, etc.)
  • Full administrative panel
    • Attendee management system
    • Conference and talk management system
    • Attendee auto-emailer
    • LaTeX program generator
    • LaTeX name badge generator
    • Attendee batch updater (for fixing form typos, etc.)
  • Fully editable HTML templates
